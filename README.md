# 🚲 Bike Sharing Analysis Dashboard

Dashboard ini menyajikan analisis data penyewaan sepeda untuk memahami pola penggunaan berdasarkan waktu, kondisi cuaca, dan hari kerja.

---

## 📁 Struktur Proyek

    submission/
    ├── dashboard/              # Aplikasi utama Streamlit
    ├── data/                   # Data Mentah
    ├── requirements.txt        # Daftar library yang digunakan
    └── README.md               # Dokumentasi proyek

---

## ⚙️ Setup Environment

Pastikan Python versi 3.9 atau lebih baru sudah terpasang.

### 1. Membuat Virtual Environment

    python -m venv venv

### 2. Mengaktifkan Virtual Environment

Windows:

    venv\Scripts\activate

Mac/Linux:

    source venv/bin/activate

---

## 📦 Instalasi Dependensi

    pip install -r requirements.txt

---

## ▶️ Menjalankan Dashboard

    streamlit run dashboard/dashboard.py
